# Microsoft 365 Identity & Messaging Administration

## Overview
This project demonstrates hands-on administration of Microsoft 365 services with a focus on identity management, messaging, collaboration, security governance, and automation using PowerShell.

It reflects real-world administrative tasks commonly performed by Microsoft 365 and cloud administrators in enterprise environments.

---

## Technologies & Services Used
- Microsoft Entra ID (Azure AD)
- Exchange Online
- SharePoint Online
- OneDrive for Business
- Microsoft 365 Admin Center
- Security & Compliance Center
- PowerShell (Microsoft Graph & Exchange Online modules)

---

## Administrative Tasks Performed
- Managed Microsoft Entra ID users, groups, and role-based access control (RBAC)
- Assigned and managed Microsoft 365 licenses
- Created and configured shared mailboxes and distribution groups in Exchange Online
- Administered SharePoint Online sites and OneDrive access policies
- Configured security and compliance settings at the tenant level
- Automated administrative tasks using PowerShell

---

## Automation & Scripting
- Used PowerShell for user and license management
- Automated repetitive administrative tasks using Microsoft Graph and Exchange Online modules
- Applied scripting to improve efficiency and consistency in Microsoft 365 administration

---

## Skills Demonstrated
- Microsoft 365 Administration
- Identity & Access Management (IAM)
- Exchange Online Administration
- PowerShell Automation
- Security & Compliance Governance
- Cloud Collaboration Platforms

---

## Project Purpose
This project was created to showcase practical Microsoft 365 administration skills and to demonstrate readiness for junior to intermediate Microsoft 365 / Cloud Administrator roles.
